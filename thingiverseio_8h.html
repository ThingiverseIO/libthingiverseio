int tvio_version ( int *  major,
int *  minor,
int *  fix 
)

Gets the version of ThingiverseIO. Useful to check if the shared library is linked correctly.

Error Codes:

  • ERR_INVALID_DESCRIPTOR C.int = 1
  • ERR_INVALID_INPUT C.int = 2
  • ERR_INVALID_OUTPUT C.int = 3
  • ERR_INVALID_RESULT_ID C.int = 4
  • ERR_INVALID_REQUEST_ID C.int = 5
  • ERR_NO_RESULT_AVAILABLE C.int = 6
  • ERR_RESULT_NOT_ARRIVED C.int = 7
    Parameters
    majorPointer to an integer that will be set to version major.
    minorPointer to an integer that will be set to version minor.
    fixPointer to an integer that will be set to version fix.
    Returns
    error
